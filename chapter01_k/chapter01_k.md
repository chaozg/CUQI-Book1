# Chapter 10: Sampling with CUQIpy-PyTorch

In this chapter, we introduce the [CUQIpy-PyTorch plugin](https://github.com/CUQI-DTU/CUQIpy-PyTorch), which allows users to combine modeling capabilities of CUQIpy with the powerful automatic differentiation and GPU acceleration features of PyTorch. The use of this plugin is illustrated through an example of solving a deconvolution BIP.
