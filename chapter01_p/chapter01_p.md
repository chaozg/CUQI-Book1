# Chapter 14: More applications and benchmarks using CUQIpy

In this chapter, we highlight the [CUQIpy User Showcase repository](https://github.com/CUQI-DTU/CUQIpy-User-Showcase) which is a collection of application problems that use CUQIpy in various BIPs. These cases were contributed by users of CUQIpy. We also highlight a [benchmark collection](https://github.com/CUQI-DTU/CUQIpy-Benchmarks) of BIPs that were built using CUQIpy for the purpose of providing a common ground for testing and comparing different sampling methods in CUQIpy and other libraries.
